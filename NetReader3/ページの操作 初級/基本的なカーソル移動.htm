基本的なカーソル移動

基本的なカーソルの移動方法を説明します。

矢印キーは、次のように移動します。

　・次の文節へ　　　→
　・前の文節へ　　　←
　・次の段落へ　　　↓
　・前の段落へ　　　↑

Ctrlと組み合わせることにより次のような移動に変わります。

　・次の文字へ　　　Ctrl + →
　・前の文字へ　　　Ctrl + ←
　・次の行へ　　　　Ctrl + ↓
　・前の行へ　　　　Ctrl + ↑

段落の先頭と末尾に移動できます。

段落とは、改行までのブロックのことを言います。
　・段落の先頭へ　　Home
　・段落の末尾へ　　End

ページの先頭と末尾にも移動できます。

　・ページの先頭へ　Ctrl + Home
　・ページの末尾へ　Ctrl + End

２０の倍数単位で、行が移動できます。

　・前の２０の倍数行へ　PageUp
　・次の２０の倍数行へ　PageDown

インテリジェントジャンプ

現在のカーソル位置の情報からころあいの区切りを、ネットリーダーが見つけ出し、この区切りに移動します。
この移動を インテリジェントジャンプと呼んでいます。
　・前の位置にインテリジェントジャンプ　Ctrl + PageUp
　・次の位置にインテリジェントジャンプ　Ctrl + PageDown

フレームが使用されたページでは、各フレームの先頭への移動ができます。

　・次のフレームへ　Ctrl + Tab
　・前のフレームへ　Ctrl + Shift + Tab

ここまでの点字コマンドをまとめました。

　・段落の先頭へ　　　　せ + 無変換 （先頭の せ）
　・段落の末尾へ　　　　ら + 無変換 （ラストの ら）
　・ページの先頭へ　　　と + Space （トップの と）
　・ページの末尾へ　　　ら + Space （ラストの ら）
　・前の２０の倍数行へ　は + Space （バックの は）
　・次の２０の倍数行へ　ね + Space （ネクストの ね）
　・前の位置にインテリジェントジャンプ　　は + 無変換 （バックの は）
　・次の位置にインテリジェントジャンプ　　ね + 無変換 （ネクストの ね）
