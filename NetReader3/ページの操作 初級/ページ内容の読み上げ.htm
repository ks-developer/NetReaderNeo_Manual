ページ内容の読み上げ

ネットリーダーにはいろいろな読み上げが用意されています。この中から、基本的な読み上げを説明します。
　・カーソル位置から読み上げ　　　　　　Alt + F10 または 右Shift
　　　この読み上げの停止　　　　　　　　Escape または Ctrl
　　　この読み上げの読み始め位置に移動　フルキー G
　・カーソルのある段落を読み上げ　　　　Alt + F9
　・カーソルのある行の読み上げ　　　　　Alt + F8
　・ページ名読み上げ　　　　　　　　　　Alt + F1
　・カーソル位置を行と列で読み上げ　　　F9
　　　フレームが使用されたページなら、カーソルのあるフレーム番号も読み上げます。
　・音声スピードの変更　　　　　　　　　F7

　点字コマンドは
　・カーソル位置を行と列で読み上げ　　　か + Space （カーソルの か）
　・カーソル位置から読み上げ　　　　　　お + Space （オールの お ）
　　　この読み上げの停止　　　　　　　　6の点 + Space
　　　この読み上げの読み始め位置に移動　よ + Space （読み始めの よ）
　・カーソルのある段落を読み上げ　　　　る + Space （以前は、ループ読みと呼んでいた）
　・カーソルのある行の読み上げ　　　　　わ + Space （ワンラインの わ）
　・ページ名読み上げ　　　　　　　　　　し + 無変換（ページの し）

その他、詳しい読み上げは、マニュアル《ページを読み上げる》を参照してください。
