お気に入りを整理する

●お気に入りの移動

お気に入りの登録場所の移動について説明します。

①プルダウンメニュー［お気に入り-お気に入りの整理］を実行します。
実行すると、追加されているお気に入りが一覧されます。

②↑や↓を使用して、移動したいお気に入りを選択します。
選択項目がフォルダの場合は、タイトルを読み上げた後に「フォルダ」と音声ガイドします。
→を押すとフォルダの内部へ移動し、←を押すと戻ります。
※補足
お気に入りを選択して Alt + F1を押すと、選択位置を音声ガイドします。
また Alt + F5を押すと、選択中のお気に入りのタイトルとURLを読み上げます。

③移動したいお気に入りを選択し、Tabを押します。

④「移動」と音声ガイドがありますので、Enterを押します。

⑤移動先が一覧されますので、↑や↓を使用して、移動先のお気に入り位置を選択します。
選択項目がフォルダの場合は、→を押すとフォルダの内部へ移動し、←を押すと戻ります。
移動を中止したい場合は、Escapeを押します。

⑤移動先が決まれば、Tabを押します。
「この上へ移動」と音声ガイドがあり、もう一度Tabを押すと「この下へ移動」と音声ガイドがあります。

⑥「この上へ移動」でEnterを押すと、選択中のお気に入りの上の位置へお気に入りを移動します。
　「この下へ移動」でEnterを押すと、選択中のお気に入りの下の位置へお気に入りを移動します。
移動を中止したい場合は、Escapeを押します。


●名前の変更

お気に入りの名前の変更方法について説明します。

①プルダウンメニュー［お気に入り-お気に入りの整理］を実行します。
追加されているお気に入りが一覧されます。

②↑や↓を使用して、名前を変更したいお気に入りを選択します。
選択項目がフォルダの場合は、タイトルを読み上げた後に「フォルダ」と音声ガイドします。
→を押すとフォルダの内部へ移動し、←を押すと戻ります。
※補足
お気に入りを選択して Alt + F1を押すと、選択位置を音声ガイドします。
また Alt + F5を押すと、選択中のお気に入りのタイトルとURLを読み上げます。

③名前を変更したいお気に入りを選択し、Tabを二回押します。

④「名前の変更」と音声ガイドがありますので、Enterを押します。

⑤お気に入りの名前の入力画面が開きますので、名前を変更します。

⑥Enterを押して、名前を変更します。
名前の変更を中止したい場合は、Escapeを押します。

※補足
名前を変更したいお気に入りを選択して、F2を押すだけでもお気に入りの名前を変更することができます。


●削除

お気に入りの削除方法について説明します。

①プルダウンメニュー［お気に入り-お気に入りの整理］を実行します。
実行すると、追加されているお気に入りが一覧されます。

②↑や↓を使用して、削除したいお気に入りを選択します。
選択項目がフォルダの場合は、タイトルを読み上げた後に「フォルダ」と音声ガイドします。
→を押すとフォルダの内部へ移動し、←を押すと戻ります。
※補足
お気に入りを選択して Alt + F1を押すと、選択位置を音声ガイドします。
また Alt + F5を押すと、選択中のお気に入りのタイトルとURLを読み上げます。

③削除したいお気に入りを選択し、Tabを三回押します。

④「削除」と音声ガイドがありますので、Enterを押します。
削除を中止したい場合は、Escapeを押します。

⑤削除の確認メッセージが表示されます。
削除しても構わないならば、Enterを押します。
削除したくない場合は、Escapeを押します。

※補足
削除したいお気に入りを選択して、Deleteを押すだけでも削除することができます。
この場合も確認メッセージが表示されます。


●フォルダの作成

フォルダを作成すると、複数のお気に入りをジャンルごとにまとめるといったことができます。
お気に入りの整理整頓にご活用ください。
ここでは、フォルダの作成方法について説明します。

①プルダウンメニュー［お気に入り-お気に入りの整理］を実行します。
実行すると、追加されているお気に入りの一覧が表示されます。

②矢印キーを使用して、フォルダの作成先のフォルダへ移動します。
※補足
お気に入りを選択して Alt + F1を押すと、選択位置を音声ガイドします。
また Alt + F5を押すと、選択中のお気に入りのタイトルとURLを読み上げます。

③Tabを四回押します。

④「新しいフォルダ」と音声ガイドがありますので、Enterを押します。

⑤新しく作成するフォルダ名の入力画面が開きます。
作成したいフォルダ名を入力してください。

⑥Enterを押すと、お気に入り一覧の終端に、フォルダが作成されます。
フォルダの作成をキャンセルしたい場合はEscapeを押します。

※補足
Ctrl + Nを押すだけでもフォルダを作成することができます。
