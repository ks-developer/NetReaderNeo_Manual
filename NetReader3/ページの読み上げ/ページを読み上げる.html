ページの読み上げ操作には、読み上げ確認したい範囲に応じた全文読みや段落読み、１行読みなどがあります。


■全文読み

　Alt + F10で 現在のカーソル位置から読み上げを開始します。
　読み上げ中は、カーソルが読み上げている場所を追いかけながら進んでいきます。
　またカーソルを操作して、読み上げ位置を自由に移動することもできます。
　読み上げを開始したカーソル位置へ戻りたい場合は、フルキー ＾（べき乗）を押します。
　読み上げを中断したいときは、Escapeを押してください。


■現在のカーソル上の文字や文章を読み上げるコマンドも豊富にあります。

  • カーソル文字の情報読み　　　　　　Alt + F3 または Ctrl + Alt + G
  • カーソル位置を行と列で読み上げ　　F9
    　　　フレームが使用されたページなら カーソルのあるフレーム番号も読み上げます。
  • カーソル行の読み上げ　　　　　　　Alt + F8
  • カーソル段落の読み上げ　　　　　　Alt + F9
  • 行頭からカーソル位置の手前までの読み上げ
    　　　PC-Talker　　　　　　　 Ctrl + Alt + H
    　　　VDM　　　　　　　　　　 Ctrl + Alt + J
  • カーソル位置から行末までの読み上げ
    　　　PC-Talker　　　　　　　 Ctrl + Alt + K
    　　　VDM　　　　　　　　　　 Ctrl + Alt + L
  • 段落読み Alt + F9
  • カーソル文字の再読み
    　　　PC-Talker　　　　　　　 Ctrl + Alt + R
    　　　VDM　　　　　　　　　　 Ctrl + Alt + G
  • カーソル文字の点字符号読み
    　　　PC-Talker　　　　　　　　Ctrl + Alt + F
    　　　VDM　　　　　　　　　　　Ctrl + Alt + X
  • カーソル文字のコード読み
    　　　PC-Talker　　　　　　　 Ctrl + Alt + I
    　　　VDM　　　　　　　　　　 Ctrl + Alt + N

■ページ読み上げの点字コマンドは、次の通りです。

  • カーソル位置を行と列で読み上げ　か + Space （カーソルの か）
  • 全文読み　　　　　　　　　　　　お + Space （オールの お）
  • １行読み　　　　　　　　　　　　わ + Space （ワンラインの わ）
  • 段落読み　　　　　　　　　　　　る + Space （以前は、ループ読みと呼んでいた）
  • 行頭からカーソル手前読み　　　　れ下がり + 無変換 ※PC-Talkerのみ
  • カーソルから行末読み　　　　　　ら下がり + 無変換 ※PC-Talkerのみ

以下は六点キー周辺に配置されているキーで、実行します。

（1の点がFキー、4の点がJキーの場合）

  • カーソル文字の再読み　　　　1の点 右隣 G
  • カーソル文字の点字符号読み　3の点 右下 X
  • カーソル文字のコード読み　　3の点 左下 Z
