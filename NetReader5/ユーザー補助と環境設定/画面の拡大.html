●ブラウザ画面の拡大

ここでは、弱視者のかたに優しい、画面表示の設定方法を紹介します。

ブラウザ画面の拡大率は次のコマンドで変更できます。
　・拡大率を上げる　　Alt + PageDown　または Ctrl + +
　・拡大率を下げる　　Alt + PageUp　または Ctrl + -
　・拡大率100%を設定　Ctrl + 0


●テキスト情報画面の拡大

ブラウザ画面の操作している箇所を表示するテキスト情報画面の拡大率を変更することができます。

プルダウンメニュー［表示-テキスト情報画面の拡大率］の中から倍率を選択して、最も見やすい大きさに設定してください。


●メニューとダイアログの拡大

プルダウンメニューの文字や、ダイアログの拡大率を変更できます。
ダイアログとは、ファイル名の入力画面や、検索文字列の入力画面など、機能を実行したときに現れる、設定画面のことです。

プルダウンメニュー［表示-メニューとダイアログの拡大率］の中から倍率を選択して、最も見やすい大きさに設定してください。
