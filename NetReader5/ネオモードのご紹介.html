1．クラシックモードとネオモード

NetReader Neoでは、従来のNetReaderと呼んでいたテキストブラウザは、クラシックモードという名称に変更しました。

クラシックモードは、従来のNetReaderⅡと同じInternet Explorerをベースとして開発されたテキストブラウザで、テキスト中心のシンプルなページを主な対象としています。

しかし、ベースのInternet Explorerの開発はすでに終了しており、インターネット上のページがInternet Explorerを対象としてない場合や、クラシックモードでは操作ができない場合も目立つようになりました。

クラシックモードでは操作できないページを操作できること、インターネットのトレンドに対応できるよう常に最新のブラウザを提供することを目的に、chromiumをベースとした新ブラウザであるネオモードを開発し追加しました。

2．各モードの開き方

クラシックモードからネオモード、あるいはネオモードからクラシックモードを開くことができます。

どちらの場合も、拡張メニュー Alt + K を開きます。

クラシックモードからネオモードを開く場合は、メニュー内の「現在のページをネオモードで開く」を実行します。

また、リンクにカーソルがある場合、拡張メニューの「リンク」-「ネオモードで開く」を実行すると、リンク先をネオモードで開きます。

ネオモードからクラシックモードを開く場合は、メニュー内の「現在のページをクラシックモードで開く」を実行します。

また、リンクにカーソルがある場合、拡張メニューの「リンク先をクラシックモードで開く」を実行すると、リンク先をネオモードで開きます。

開いているモードを確認するには、メニュー「ヘルプ」-「バージョン情報」から確認する方法と、PC-Talkerのウインドウ名読み( Ctrl + Alt + 1 )で確認する方法があります。

3.タブブラウザについて

ネオモードは、タブブラウザと呼ばれるブラウザで、起動しているネオモードに複数のページを開くことができ、簡単に切り替えることが可能となっています。

クラシックモードの場合は、一つのブラウザに一つのページのみ開くことしかできませんでした。
このため、クラシックモードでページを切り替えるには Alt + Tabキーでアプリを切り替える必要がありました。

ネオモードのタブコマンド

  • 新しいタブを開く Ctrl + T
  • 新しいタブでリンク先を開く Ctrl + Shift + Enter
  • 次のタブへ切替　Ctrl + Tab
  • 前のタブへ切替　Ctrl + Shift + Tab
  • 現在のタブを閉じる Ctrl + 4
  • タブウインドウの一覧 Ctrl + F6
  • タブウィンドウ一覧の読み上げ Ctrl + Shift + F6

4.テーブルの移動について

テーブルに移動した場合、クラシックモードと同じように Insertキーでテーブルモードに切り替えることができます。

テーブルモード中は、上下キーで行の移動、左右で列を移動します。

クラシックモードでは、最終行で下矢印キー、または先頭行で上矢印キーを押すと
テーブルの外に移動できませんでしたが、ネオモードではテーブルの外に移動します。
