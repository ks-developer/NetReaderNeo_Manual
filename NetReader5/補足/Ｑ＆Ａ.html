ネットリーダー　ネオモードに、よく寄せられる質問にお答えします。

※新しい順に記述しています。

質問１１ PDFファイルを変換して開こうとすると、パスワードを要求されます。

お答え
PDFファイルには、文書を開くパスワードの他に、印刷や編集を禁止する権限パスワードが設定されている場合があります。
ネットリーダー、およびマイブックのPDF変換機能は権限パスワードに対応しておらず、開くことができません。
大変申し訳ありませんが、権限パスワードが設定されたPDFにつきましては、Acrobat Readerを使用して開くようお願いいたします。


質問１０ YouTubeアドインで動画プレーヤーの操作を行うことができません。

お答え
「環境-環境のクリーンアップ-キャッシュされたファイルの削除」を行う必要があります。


質問９ 楽天で買い物かごにアクセスすると「Bad request」というメッセージで、買い物かごに進めません。

お答え
解消するには、「環境-環境のクリーンアップ-Cookieの削除」を行うと解消される場合があります。


質問８ Googleにログインすることができません。

お答え
Googleにログインできない場合があることを確認しています。
ログイン時に、プルダウンメニュー[環境-スマートフォン用のページを開く]を実行し、ログインすることにより、回避することが可能です。


質問７ ボタンを押しても反応しません。

お答え
ボタンによってはマウスクリックのみ反応するページがあります。
このような場合は、@(アットマーク)を押すと、マウスクリックコマンドが実行され、反応する場合があります。


質問６ ネオモードで移動も読み上げもできない箇所があります。

お答え
ネオモードでは、画面上に表示されていない隠しテキストに移動、読み上げることができません。
このような場合は、エンターキーを押すと、隠しテキストが表示され、移動できる場合があります。
特に質問と答えが用意されているFAQ形式の場合に多く見受けられますが、答えテキストが表示されておらず、
質問文でエンターキーを押すと、答えテキストが表示され、移動できる場合があります。


質問５ Enterキーを押しても操作できません。(Googleドライブのファイルなど)

お答え
ネオモードで、Enterキーを押しても目的の操作を実行できない場合があります。
このような場合は、Chromiumモード Ctrl + Insertに切り替えて操作するか、@(アットマーク)を押すと実行できる場合があります。
また、Googleドライブ内のファイルを開く場合は、目的のファイル選択を選択し、Enterを押した後、ダブルクリックのように@(アットマーク)を素早く2回押すと開くことができます。


質問４ Flashを再生することができません。

お答え
Flashのサポートは2020年12月31日にサポートを終了し、再生することはできなくなりました。
Flash Playerをインストールされている場合、直ちにアンインストールされることをお勧めいたします。


質問３　ページに設定されたキーが使えません。

お答え
ページによっては、独自のショートカットキーが設定されている場合があります。
このようなページ独自のショートカットキーを使用するには、次の手順で操作を行います。

  1. Chromiumモード Ctrl + Insert に切り替えます。
  2. Ctrl + Space を押してPC-Talkerのナビゲーションキーをオフにします。
  3. ページに設定されているキー操作を行います。
  4. 操作が完了すれば、Ctrl + Space を押してPC-Talkerのナビゲーションキーをオンにします。
  5. Escapeで Chromiumモードを解除します。

質問２　パスワードや、入力エディットに文字を入力してもページに反映されません。

お答え
ページによっては、セキュリティ強化のため、ページ内のエディットのみ入力が許可されている場合があります。
(Twitterやfacebookの投稿エディットなど)
ページ内のエディットのみ入力を許可されたページは、エディット上でEnterや、F2を押して入力画面を開いて文字を入力してもページに反映されません。
このようなページのエディットに入力を行うには、次の手順で文字を入力します。

  1. エディット上で　Chromiumモード Ctrl + Insert に切り替えます。
  2. ← を押して入力モードに切り替えます。
  3. 文字を入力します。
  4. Ctrl + Insertで Chromiumモードを解除します。

質問１　クラシックモードでは動作するが、ネオモードでは動作しないページがあります。

お答え
ページがInternet Explorerのみ対応している場合があります。
ネオモードでは、このようなページは対応できないため、クラシックモードで操作していただく必要があります。
