ネオモードは、タブブラウザと呼ばれるブラウザで、起動しているネオモードに複数のページを開くことができ、簡単に切り替えることが可能となっています。

それでは、タブに関連した操作を説明します。

  • 新しいタブを開く Ctrl + T
    　新しいタブでホームに登録されたページを開きます。
  • 新しいタブでリンク先を開く Shift + Enter
    　カーソル位置がリンクの場合、リンク先を新しいタブで開きます。
  • 次のタブへ切替　Ctrl + Tab
    　タブを複数開いている場合、次のタブに切り替えます。
  • 前のタブへ切替　Ctrl + Shift + Tab
    　タブを複数開いている場合、前のタブに切り替えます。
  • 現在のタブを閉じる Ctrl + F4
    　操作中のタブを閉じます。
    　タブが一つの場合、ネットリーダーを終了します。
  • タブウインドウの一覧 Ctrl + F6
    　タブ一覧画面を開きます。
    　タブ一覧画面では、↑↓で開いてるタブを選択し、Enterでタブを切り替えることができます。
    　また、一覧画面でタブを選択し、Tabを押してタブを閉じることもできます。
  • タブウィンドウ一覧の読み上げ Ctrl + Shift + F6
    　開いているタブの数と、ページタイトルを読み上げます。
